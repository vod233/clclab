GitHub Path · 学习路线

从 git init 到第一次 开源贡献。

一条循序渐进的学习路线，把 GitHub 的核心能力拆成五个可练习的阶段。 每一步都对应一个可运行的小项目，让你在做中学。

01
Git 基础

理解 commit / branch / merge，掌握 add · commit · push · pull 的最小循环。

02
仓库与分支

创建仓库、配置 .gitignore、用 feature branch 隔离每一次改动。

03
PR 协作

打开第一个 Pull Request，写好 description，礼貌地 review 别人的代码。

04
Actions 自动化

用 GitHub Actions 跑测试、构建、部署，配置 CI 让项目自己跑起来。

05
开源贡献

挑一个你喜欢的项目，从 good first issue 开始，给世界留下你的第一行代码。
